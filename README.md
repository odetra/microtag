This shell script generates a static booru-like website. You can see a demo <a href="https://odetra.github.io/microtag-demo">here</a>.

It may be useful for creating galleries in platforms like neocities.org or github.io, where you don't have backend access.
This script is for small galleries, like hundreds, maybe a few thousands of files. For a lot of thousands of files it will (may, I did not test it) be slow.
Remember: it's just bash. A dirty and quick makeshift solution.
